## 4. 반응형 전략

### Breakpoints (Tailwind 기준)
```
sm: 640px   — 모바일 가로
md: 768px   — 태블릿
lg: 1024px  — 데스크탑
xl: 1280px  — 와이드
```

### 검색 결과 페이지 레이아웃
- **lg+**: `grid grid-cols-[400px_1fr]` (좌측 리스트 고정폭 + 우측 지도)
- **md~lg**: 상단 지도(50vh) + 하단 스크롤 리스트
- **~md**: 탭 전환 (지도/리스트) + 바텀시트 상세

### 필터바
- **md+**: 가로 배치 (한 줄)
- **~md**: 필터 아이콘 버튼 → 클릭 시 바텀시트 or 모달

## 5. 상태 관리

- **서버 상태**: TanStack Query (React Query) — API 응답 캐싱
- **클라이언트 상태**: Zustand (검색 필터, 지도 뷰포트 등)
- **URL 상태**: Next.js searchParams (검색 조건을 URL에 반영 → 공유 가능)

## 6. 지도 연동 (Kakao Map)

### 사용 API
- Kakao Maps JavaScript SDK: 지도 렌더링, 마커, 폴리라인
- Kakao Local API: 주소 검색, 자동완성
- Kakao Mobility API: 대중교통 경로 탐색

### 지도 컴포넌트 구현
- `react-kakao-maps-sdk` 패키지 활용
- 또는 Kakao Maps SDK를 직접 `useEffect`로 로드
